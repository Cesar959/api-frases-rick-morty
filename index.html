Sobre

Interface de Programação de Aplicações (português) cuja sigla API provém do Inglês Application Programming Interface, é um conjunto de rotinas e padrões estabelecidos por um software para a utilização das suas funcionalidades. Com o intuito de aprender como desenvolver uma api utilizando o Slim framework

Introdução

API conta com 2 funcionalidades básicas: obter todos os registros e selecionar um unico registro. Até o momento não foi implementado um sistema de autenticação para utilizar a API, possibilitando assim a utilização pública da mesma.

Tecnologias

  • Slim framework
  • PHP
  • Composer
  • Json

Metodos

GET - utilizado para pegar os dados

Rotas

É a mesma rota para GET e GETid que utiliza o 'id' do registro como parâmetro. Alem disso o unico serviço disponivel até o momento é "frases".

Traz todos os registros

GET

http://csantosalmeida.rf.gd/api-frases-rick-morty/frases

Traz um registo específico (o caso em questão, vai trazer os dados referente ao 'id' 2)

GET

http://csantosalmeida.rf.gd/api-frases-rick-morty/frases/2

Mensagens de Status

200 : Requisição foi feita com sucesso.

404: Falha na requisição, possivelmente por falta de envio de dados ou os dados solicitados não existe

400: Recuso não localizado

500: Erro interno do servidor.
